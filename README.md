# ImageToolBox
Image Process Tools  

影像處理工具主要包含  
1.像素點的轉換 (Point Processing)  
2.空間卷積(Space Filter)  
3.區域分割(Segmention)  
4.雜訊添加(Noise)  
5.特徵擷取(Feature Extraction)  
6.風格轉換(Style Space Filter)  
7.頻率域轉換(Frequency Domain)  
8.形態學處理(Morphology)  

---
#程式實作上使用之Design Pattern技巧
1. Template Pattern。將Point Processing與Space Filter的處理抽象化為Template。
